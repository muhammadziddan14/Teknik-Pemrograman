# Teknik Pemrograman - Kumpulan Tugas Praktikum Java

Repositori ini berisi dokumentasi dan solusi kode untuk mata kuliah Teknik Pemrograman. Proyek ini mencakup pemahaman konsep dasar Java mulai dari tipe data, variabel, hingga manipulasi konstanta.

## Daftar Tugas

### 1. Data Types (Solution.java)
Program untuk menentukan kategori tipe data integer yang paling efisien (`byte`, `short`, `int`, atau `long`) untuk menyimpan angka masukan. Program ini sudah dilengkapi dengan penanganan *exception* untuk angka yang melebihi kapasitas 64-bit.

### 2. Constants (Constants.java & Constants2.java)
Latihan penggunaan kata kunci `final` di Java.
* **Constants.java**: Menunjukkan penggunaan konstanta di dalam metode `main`.
* **Constants2.java**: Menunjukkan penggunaan konstanta statis (`static final`) yang dapat diakses oleh seluruh bagian kelas.

### 3. Floating Point (FloatingPoint.java)
Eksperimen dengan tipe data bilangan pecahan. Tugas ini mendemonstrasikan bagaimana Java menangani pembulatan angka menggunakan fungsi `Math.round()` dan perilaku tipe data `double`.

### 4. Convert Data Type (ConvertDataType.java)
Latihan teknik *type casting* di Java. Mendemonstrasikan cara mengubah satu tipe data ke tipe data lainnya (misalnya dari `double` ke `float` atau `int`) dan memahami risiko kehilangan presisi data.

### 5. Operator Challenge (OperatorChallenge.java)
Tugas mengenai penggunaan operator aritmatika dan logika untuk menyelesaikan perhitungan spesifik, serta pemahaman urutan presedensi operator dalam ekspresi matematika.
